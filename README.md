# AsciiTab (Catppuccin Startpage) 🚀

AsciiTab is a high-performance, keyboard-centric browser startpage designed for power users who love ASCII aesthetics and terminal-like workflows.

## ✨ Key Features

- **⌨️ Command-Driven Workflow**: Navigate and search the web instantly using prefixes (`!gh`, `!yt`, etc.) or manage settings with `@` commands.
- **🌍 Minimalium Mode**: A dedicated minimalism manager with 3 distinct modes to hide all distractions and focus purely on your ASCII art and search.
- **🎨 ASCII Hero Editor**: Built-in editor for custom ASCII art with zoom/size controls and instant preview. Includes presets like **Tux**, **Earth**, and **Dragon**.
- **🐈 Catppuccin Support**: Fully themed with Catppuccin's vibrant palette (Mocha, Macchiato, Frappe, and Latte).
- **🧩 Integrated Widgets**: Optional sidebar with Tasks, Notes (Memo), Pomodoro Timer, Weather, Media Player, and Real-time System Stats.
- **🔒 Privacy First**: 100% local storage. No tracking, no accounts, no cloud.

## 🚀 Commands & Shortcuts

### System Commands
| Command | Action |
| :--- | :--- |
| `@s` | Open Settings Modal |
| `@min` / `@minimalium` | Open Minimalium Manager |
| `@ae` / `@ascii` | Open ASCII Editor |
| `@theme [name]` | Switch theme (mocha, frappe, macchiato, latte) |
| `@h` | Open Help Menu |
| `@c` | Clear Command Bar |
| `@wm` / `@widgets` | Open Widget Management |
| `@sidebar [on/off]` | Toggle the left sidebar |
| `@reset` | Factory Reset (Wipes all local data) |

### Global Shortcuts
| Key | Action |
| :--- | :--- |
| `Ctrl + K` | Focus Search Bar |
| `Ctrl + ,` | Open Settings |
| `Ctrl + M` | Open Minimalium Manager |
| `ESC` | Close any open Modal |

## 📐 Minimalism Modes (Minimalium)

Accessible via `@min` or the Settings menu, Minimalium lets you strip down the UI:
1. **Art + Clock + Search**: Keeps the core essentials.
2. **Art + Search**: Removes the time for extra focus.
3. **Art Only**: Total focus. (Tip: Use `Ctrl + ,` or `Ctrl + M` to exit this mode).

## 🛠️ Customization

### ASCII Art Presets
Choose from several built-in designs:
- `Default`: The classic ASCIITab logo.
- `Tux`: The iconic Linux mascot.
- `Earth`: A detailed ASCII globe.
- `Minimal`: A sleek, abstract design.
- `Dragon`: A fierce traditional ASCII dragon.

### Search Engines
Search suggestions and default engine can be toggled in the Settings (`@s`).
